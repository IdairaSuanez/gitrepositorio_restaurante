# Restaurante Los grandes sabores

Sitio web desarrollado para el restaurante "Los grandes sabores".

## Archivos del proyecto
- **index.html**: página principal del restaurante
- **style.css**: hoja de estilos
- **contacto.html**: formulario de contacto para clientes
- **menu.html**: carta y menú del restaurante

## Tecnologías utilizadas
- HTML5
- CSS3

## Control de versiones
Proyecto gestionado con Git y GitHub.
